Bayesian quadrature object¶

class bayesian_quadrature.bq.BQ(R, S, gamma, ntry, n_candidate, R_mean, R_var, h=None, w=None, s=None)[source]¶

Bases: object

Estimate a likelihood function, S(y|x) using Gaussian Process regressions, as in Osborne et al. (2012):

  1. Estimate S using a GP
  2. Estimate log(S) using second GP
  3. Estimate delta_C using a third GP

References

Osborne, M. A., Duvenaud, D., Garnett, R., Rasmussen, C. E.,
Roberts, S. J., & Ghahramani, Z. (2012). Active Learning of Model Evidence Using Bayesian Quadrature. Advances in Neural Information Processing Systems, 25.
Cw(gp)[source]¶

The variances of our posteriors over our input scale. We assume the covariance matrix has zero off-diagonal elements; the posterior is spherical.
